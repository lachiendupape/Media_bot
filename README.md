# Media Bot

This project is a custom agent designed to interact with a home media server stack. It can communicate with various services to retrieve information and request new media.

## Features

*   **Tautulli**: Get status of media playback.
*   **Radarr**: Add movies.
*   **Sonarr**: Add TV series.
*   **Lidarr**: Add music.

## Development Plan

1.  **Project Scaffolding**: Create the basic project structure.
2.  **Configuration**: Implement a way to manage API keys and URLs.
3.  **API Wrappers**: Create Python classes to interact with each service's API.
4.  **Bot Logic**: Implement the core logic for the bot to understand commands and interact with the API wrappers.
5.  **Interface**: Create a simple command-line interface (CLI) to interact with the bot.
